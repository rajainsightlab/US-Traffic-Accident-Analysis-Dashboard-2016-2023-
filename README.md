
🚀 US Traffic Accident Analysis Dashboard (2016–2023)

🛠️ Tools Used:
Power BI | Data Cleaning | DAX | Data Visualization

![US Traffic Accident Analysis Dashboard ](Screenshot 2026-05-01 171034.png?raw=true.

I recently built an interactive data analytics dashboard to analyze over 3 million traffic accident records across the United States. The objective was to transform raw data into meaningful insights and uncover patterns related to vehicle types, severity, and accident trends.

📊 Key Highlights:
• Cars contribute to ~79% of total accidents
• Truck accidents show higher severity levels
• Most accidents occur during clear weather & dry road conditions
• Peak accident times: morning & evening rush hours
• Majority cases involve minor damage, but injuries & fatalities remain significant


🎯 Outcome:
This project enhanced my ability to analyze large datasets, build interactive dashboards, and present insights through effective data storytelling.
💡 Insight:
Accidents are driven more by human behavior and traffic patterns than external conditions.
